# Canvas
Welcome to Canvas: a final project for FHSCS sophomore year

This project is meant to focus on how animation, time, and moving visuals are manipulated within a simplified context
Therefore, I chose to work with Java GUI to demonstrate the generation of simple geometric shapes with Robert Penner's easing functions:
https://github.com/jesusgollonet/processing-penner-easing/tree/master/src

The majority of the visuals are created by me.
